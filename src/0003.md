# 类型注解

作用：相当于强类型中的类型声明，对变量起到约束作用

语法：（变量/函数）：type


# 原始类型

``` javascript
let bool: boolean = true
let num: number | undefined | null = 123
let str: string = 'abc'
str = 123 // 报错，变量的数据类型是不可以互相赋值改变的
```
